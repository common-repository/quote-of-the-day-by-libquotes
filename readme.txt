== Description ==
This plugin adds a Quote of the Day widget to your WordPress blog. Each day a new handpicked quotation will appear on the sidebar of your page.

Note: This plugin connects to (libquotes.com) to generate the quote of the day. See our [terms of use](https://libquotes.com/terms) for more information.

== Installation ==
Upload the plugin to your blog, Activate it, then go to `Appearance` -> `Widgets` and drag the widget to the sidebar.
